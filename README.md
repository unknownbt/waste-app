## 🌍 About The Project

**EcoTrade** is a full-stack web and mobile platform that bridges the gap between **households** (who want to sell recyclable waste) and **recycling plants** (who want to buy it).

Instead of throwing recyclable items into landfills, users can list their waste, get fair pricing, and connect with the nearest recycling plant — all in one platform.

---

## ✨ Features

- 🏠 **Household Dashboard** — List recyclable items with photos, weight, and pickup location
- 🏭 **Recycling Plant Dashboard** — Browse available waste listings and place buy requests
- 📍 **Google Maps Integration** — Find nearest recycling centers based on user location
- 🔥 **Real-time Updates** — Firebase-powered live notifications and status tracking
- 📱 **Responsive Design** — Works seamlessly on mobile and desktop
- 🔐 **Authentication** — Secure login/signup for both user roles

## 🚀 Getting Started

### Prerequisites
Make sure you have these installed:
- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)
- [Git](https://git-scm.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/unknownbt/waste-app.git

# 2. Navigate into the project
cd waste-app

# 3. Install dependencies
npm install

# 4. Create a .env file and add your keys
MONGO_URI=your_mongodb_connection_string
FIREBASE_API_KEY=your_firebase_key
GOOGLE_MAPS_API_KEY=your_maps_key

# 5. Start the development server
npm start
```
